A brief intro to Node.js

Node.js is a JavaScript interpreter that runs outside of the browser.

It can be ran locally on our computer, on the server etc.

We can use it to achieve the same results with C# or Java, but... with JavaScript.

All of the tools we will see use Node.js

JavaScript compilers

JavaScript version A → JavaScript version B

  getName() {
    return this.name;
  }

var _createClass = function () { function defineProperties(target, props) { for (var i = 0; i < props.length; i++) { var descriptor = props[i]; descriptor.enumerable = descriptor.enumerable || false; descriptor.configurable = true; if ("value" in descriptor) descriptor.writable = true; Object.defineProperty(target, descriptor.key, descriptor); } } return function (Constructor, protoProps, staticProps) { if (protoProps) defineProperties(Constructor.prototype, protoProps); if (staticProps) defineProperties(Constructor, staticProps); return Constructor; }; }();

function _classCallCheck(instance, Constructor) { if (!(instance instanceof Constructor)) { throw new TypeError("Cannot call a class as a function"); } }

var Person = function () {
  function Person() {
    _classCallCheck(this, Person);
  }

  _createClass(Person, [{
    key: "getName",
    value: function getName() {

    }
  }]);

  return Person;

Today we will deal with Babel.

Babel.js

Babel is a JavaScript compiler written in Node.js.

Babel will do two things for us.

  1. Transform ES6 to ES5
  2. Transform JSX to JavaScript

Let's take it for a spin at https://babeljs.io.

JavaScript bundlers

a.js + b.js → ab.js

a.js

export var lol = 'lol2'

b.js

import * as a from './a';

var lol = 'lol1';

console.log(lol);
console.log(a.lol);

ab.js

var lol$1 = 'lol2';
var lol$$1 = 'lol1';
console.log(lol$$1);
console.log(lol$1);

Rollup.js

Rollup.js is a JavaScript bundler written in Node.js.
